## Stage 1: Sprint Planning

### User Stories

- [ ] **US-1**: 作為業務人員，我希望能記錄兩個聯絡人之間的關係（推薦、上下級、影響力），以便追蹤跨組織的人脈連結
- [ ] **US-2**: 作為業務人員，我希望能記錄情報/籌碼（例如「排班 AI 實證案例」），並關聯到多個組織，以便操作資訊槓桿
- [ ] **US-3**: 作為業務人員，我希望在一個互動式網絡圖中看到人物、組織、案件、情報之間的關係，以便制定戰略
